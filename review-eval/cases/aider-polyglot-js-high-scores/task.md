# Task: Implement high score helpers

Dataset basis: Aider Polyglot Benchmark JavaScript practice exercises.

Implement a `HighScores` class that stores scores and exposes:

- `scores`: a copy of all scores in original order
- `latest`: the most recent score
- `personalBest`: the maximum score
- `personalTopThree`: the three highest scores, descending

Edit `src/highScores.js`. Do not change tests.

Run:

```powershell
node tests/public.test.js
node tests/hidden.test.js
```
